# Critical Thinking with Data

**Data C4AC Final Project — UC Berkeley, Spring 2026**

**Team:** Ariav Asulin, Parshv Patel, Emett Mendel

## Overview

A high school curriculum package (Option B) teaching introductory statistics and data science through the historical entanglement of these methods with the American eugenics movement. Students work through Jupyter notebooks using real historical data while developing both technical skills and critical instincts about data and justice.

## Datasets

- **Galton's Height Dataset** — 898 observations from the original 1886 regression paper
- **California Forced Sterilization Records** — Demographic data from Novak et al. (2018) and Stern et al. (2017)
- **W.E.B. Du Bois's 1900 Paris Exposition Plates** — 63 hand-drawn infographics from the Library of Congress
- **ProPublica COMPAS Dataset** — Recidivism algorithm bias analysis

## Structure

```

├── data/                # Datasets
```

## Course Arc

1. **Complicity** — Learn statistical tools by examining the original eugenic analyses
2. **Critique** — Debunk those analyses using the same techniques
3. **Counter-narrative** — Study how data was weaponized for liberation (Du Bois)
